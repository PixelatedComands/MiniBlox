
# Miniblox Scripts
A collections of scripts for miniblox created by PixelatedComands
I was bored & randomly found Miniblox so I whipped up a quick script. Perhaps I'll make another one?

## Chat Manager

```javascript
Features:
  - /chatmanager - a way to easily customize all included in the script

  - Custom Chat Format - custom positions and colors of the player's chat message
  - Custom Join/Leave Messages - quickly format and make join & leave messages
  - [item] & [i] Support - share your items through chat by saying '[item]' or '[i]' (toggleable)
  - Spam Protection - prevent players from saying the same thing twice in a row (toggleable)

  - /clearchat - a way to clear chat of any pesky messages on command 

All features (except /clearchat) included in a simplistic and easy-to-use GUI.

```
